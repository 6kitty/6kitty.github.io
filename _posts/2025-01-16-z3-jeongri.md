문제를 조건에 맞게 해결할 때 사용하는 라이브러리 

```python
from z3 import *
s = Solver()
```

일단 `Solver()`로 인스턴스 생성 

```python
x = Int('x')
s.add(x > 0)
s.add(x < 10)
```

`add()`로 제약 조건 추가 

```python
result = s.check()
```

`check()`로 해당 조건으로 문제가 해결되는지 판단  
sat(가능) unsat(불가능) unknown(몰라) 

```python
if s.check() == sat:
    m = s.model()
    print(m[x])
```

sat일 시(가능하다면) `model()`로 해 반환 

추가 기능은 코드로 한 번에 정리 

