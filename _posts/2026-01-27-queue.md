# 큐 

큐 하면 일단 귀신 같이 나와야 하는 것이 선입선출이다. 데이터가 입력되는 순대로 바로 처리하고 싶을 때 효율적인 자료구조이다. 이 큐의 시간복잡도는 `O(1)`이기 때문이다. 

파이썬 : `from collections import deque`를 사용한다. 

```py
from collections import deque

queue = deque()
queue.append(1) # [1]
queue.append(2) # [1, 2]
queue.append(3) # [1, 2, 3]
queue.popleft() # [2, 3]
queue.popleft() # [3]
queue.popleft() # []
```

java : `Queue<String> queue = new LinkedList<>();`를 사용한다. 

```java 
Queue<String> queue = new LinkedList<>();
queue.offer(1); // [1]
queue.offer(2); // [1, 2]
queue.offer(3); // [1, 2, 3]
queue.poll(); // [2, 3]
queue.poll(); // [3]
queue.poll(); // []
```
