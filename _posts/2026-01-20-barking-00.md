# 환경설정 

```cpp
#include <bits/stdc++.h>
```

코테에서 많이 쓰는 라이브러리를 bits 폴더 안에 넣는다는 코드이다. 이 헤더는 gcc에만 있기 때문에 Visual Studio 2017/2019에서는 따로 폴더에 넣어줘야 한다. 
몇몇 시험장에서는 위 헤더를 못쓰게 한다고 한다. 아래 대표 헤더들은 외우자. 

```cpp
#include <iostream>
#include <string>
#include <map>
#include <set>
#include <stack>
#include <vector>
#include <functional>
#include <algorithm>
```
