# 바킹독의 실전 알고리즘 : 백트래킹 

## 알고리즘 설명 

현재 상태에서 가능한 모든 후보군을 따라 들어가며 탐색하는 알고리즘
이라고 한다. 

## 연습 문제 1 : N과 M

```cpp 
#include <bits/stdc++.h>
using namespace std;

int n,m;
int arr[10];
bool vis[10];

void func(int k){
    if(k==m){
        for(int i=0;i<m;i++)
            cout<<arr[i]<<' ';
        cout<<'\n';
        return;
    }
    for(int i=1;i<=n;i++){
        if(!vis[i]){
            arr[k]=i;
            vis[i]=1;
            func(k+1);
            vis[i]=0;
        }
    }
}

int main(void){
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin>>n>>m;
    func(0);
}
```

내가 풀다가 안되겠어서 풀이 봤다. 이전 시간에 배운 재귀와도 살짝 닮아있다. 이 형태가 백트래킹의 전형적인 구조라고 한다.
