# 수식의 괄호 쌍이란? 

코테에서 한 번쯤 봤을 법한 문제이다. 

```bash
(())()()
))(()
()())(()
```

위처럼 있다고 할 때 괄호 쌍이 모두 있는지 판단하는 문제이다. 일단 안쪽부터 짝 맞추기를 해본다면 첫번째 입력만 맞다는 것을 알 수 있다. 괄호가 여러 종류일 때는 단순히 괄호 개수만으로 판별하기 어렵다. 

```bash 
({(){}})
```

이러한 유형은 스택을 사용한다면 빠른 시간복잡도로 풀어낼 수 있다.
